# Domain Framework — Asufarma Edu Content Production

## Overview

This framework governs the creation of educational Instagram content for Asufarma Manipulacao,
a compounding pharmacy in Salvador, BA. Content must be scientifically grounded, brand-consistent,
ANVISA-compliant, and optimized for organic engagement.

---

## Step-by-Step Content Creation Methodology

### Step 1 — Research
- Validate the core health claim using at least one credible source (PubMed, ANVISA, CFQ/CFF)
- Identify common misconceptions about the topic (these become hooks)
- Map the audience pain point: what does the person FEEL before finding this content?
- Check for recent trending conversations on the topic (Reels audio, Twitter/X, meme formats)

### Step 2 — Ideation
- Generate 3-5 angle options for the same topic (see Angle Types below)
- Select the angle based on: audience moment, content calendar gap, format fit
- Define the single takeaway the reader should leave with (one sentence max)
- Draft the hook (slide 1 or Reel opening) before anything else

### Step 3 — Angle Selection
- **Myth-busting:** "Você acredita que [crença errada]? Deixa eu te explicar..."
- **Listicle:** "X coisas que [beneficio/problema]"
- **Problem-Solution:** Lead with the pain, resolve with the produto/informacao
- **Data-driven:** Open with a stat or study finding
- **Story-based:** Relatable scenario (meme style) transitioning to education

### Step 4 — Content Creation
- Write all slides before designing — copy is primary, design is secondary
- Slide 1: Hook only — no explanation yet
- Slide 2-N: Each slide = one idea. Never two ideas per slide
- Last slide: CTA + disclaimer (if applicable) + Asufarma branding
- Caption: Expand the hook, add context, close with CTA and hashtags

### Step 5 — Review
- Run ANVISA compliance checklist (see below)
- Check disclaimer decision tree (see below)
- Validate brand voice alignment (tone-of-voice.md)
- Score against quality criteria (quality-criteria.md)
- Minimum passing score: 7/10 per criterion

## Format Selection Criteria

| Signal | Recommended Format |
|---|---|
| Complex topic needing explanation | Carousel (6-10 slides) |
| Quick tip or single fact | Carousel (3-4 slides) or Stories |
| Trend/meme integration | Reel (15-30s) |
| Product launch or seasonal push | Reel + Stories |
| Step-by-step routine | Carousel or Reel |
| Evergreen reference content | Carousel (save-worthy) |

## Topic Categorization

### Seasonal
- Verão: protetor solar, hidratação, pele
- Outubro Rosa / Novembro Azul: prevenção, exames
- Festas: sono, imunidade, ressaca (manipulados funcionais)
- Volta às aulas: imunidade infantil, foco, vitaminas

### Evergreen
- Sono e melatonina
- Colageno e pele/articulações
- Imunidade e microbiota (Corebiome)
- Hormônios femininos (Ormonelle)
- Performance e creatina

### Trending
- Novos estudos de ingredientes populares
- Memes de saúde virais
- Crossover com cultura pop baiana / nacional

### Product-Specific
- Sempre integrar o produto ao contexto educativo, nunca ao contrário
- Produto surge como solução, não como tema principal

## ANVISA Compliance Checklist

- [ ] Nenhuma promessa terapêutica explícita ("trata", "cura", "previne doença")
- [ ] Uso de linguagem de apoio: "pode contribuir para", "auxilia", "é associado a"
- [ ] Disclaimer "Consulte seu médico/prescritor/farmacêutico" presente quando necessário
- [ ] Não comparar diretamente com medicamentos de referência
- [ ] Não fazer afirmações de eficácia sem base científica
- [ ] Não usar antes/depois de forma inadequada para produtos manipulados
- [ ] Nomes de ativos usados corretamente (não confundir com denominações de medicamentos)

## Disclaimer Decision Tree

```
O conteúdo menciona ou implica uso interno de substância ativa?
  ├── SIM → Incluir: "Consulte seu médico, prescritor ou farmacêutico"
  └── NÃO → É um produto dermocosmético de uso tópico apenas?
               ├── SIM → Disclaimer OPCIONAL (boa prática, mas não obrigatório)
               └── NÃO → Avaliar caso a caso com farmacêutico responsável
```

Produtos que SEMPRE exigem disclaimer: Melatonina, Creatina, Corebiome, Ormonelle, hormônios
Produtos que geralmente DISPENSAM: hidratantes, protetor solar manipulado, sérum facial
